**使用流程**:
```bash

```

## 💡 经验教训

1. **AI 失忆问题**: 必须用 Hook 机制持久化任务信息到文件
2. **代码写错地方**: 小码容易把工作区和正式项目搞混，TASK_CURRENT.md 必须明确项目地址
3. **Docker 权限**: 提前配置好 docker 组，避免后续麻烦
4. **密码验证**: 本地开发不需要强密码验证，给默认值更方便
5. **本地 vs Docker**: 开发者用本地运行，生产环境用 Docker
