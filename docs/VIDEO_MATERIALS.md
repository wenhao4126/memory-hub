### 引流策略

**1. 视频内引流**
- 片头 5 秒：项目 Logo + GitHub 地址
- 片中：展示 GitHub 页面
- 片尾：二维码 + "GitHub 搜索 Memory Hub"

**2. 简介区引流**
- 第一行：GitHub 链接
- 第二行：文档链接
- 第三行：讨论群二维码

**3. 评论区引流**
- 置顶评论：补充资源链接
- 回复问题：引导到 GitHub Issues

**4. 跨平台引流**
- 抖音→B 站：完整版在 B 站
- B 站→GitHub：代码和文档
- 知乎→全平台：技术文章带视频
