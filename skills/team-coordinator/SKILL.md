# SKILL.md - 团队协调者

> 傻妞的团队协调 Skill

---

## 激活条件

当用户（憨货）提出以下类型需求时激活：
- 复杂任务需要多个步骤
- 任务涉及多种能力（搜索+分析+写作）
- 明确提到"叫XX来干"、"让团队处理"
- 任务明显需要多人协作

---

## 执行流程

### Step 1: 需求分析
1. 听懂憨货到底想要什么
2. 判断任务复杂度
3. 决定是否需要协调多个智能体

### Step 2: 任务拆解
将大任务拆成子任务：
```
大任务：写行业报告
├── 子任务1：搜集数据（研究员）
├── 子任务2：分析趋势（分析师）
├── 子任务3：撰写正文（写手）
└── 子任务4：审核校对（傻妞自己）
```

### Step 3: 分配执行
1. 根据任务类型选择对应智能体
2. 创建智能体并派任务
3. 设置超时时间和优先级
4. 记录任务状态

### Step 4: 监控进度
- 定期询问进度
- 处理卡壳问题
- 必要时介入支援

### Step 5: 结果整合
1. 收集所有子任务结果
2. 去重、排序、优化
3. 整合成最终输出
4. 添加傻妞的总结和建议

### Step 6: 交付汇报
向憨货汇报：
- 任务完成情况
- 关键成果摘要
- 详细内容（如有需要）
- 后续建议

```
2. 让分析师做对比...
3. 让写手整理成文...

```

## 注意事项

1. **不要过度协调**：简单任务自己干，别折腾团队
2. **及时沟通**：进度有变化立即告诉憨货
3. **质量保证**：重要结果必须审核
4. **灵活应变**：计划赶不上变化时及时调整
5. **记录经验**：每次协调后更新 MEMORY.md
